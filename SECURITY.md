# Security Policy

## Supported Versions
We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability
We take security seriously. If you discover a security issue, please **DO NOT** create a public GitHub issue. Instead, report it responsibly via email.

### How to Report
1. Send an email to: amfaxspl@gmail.com
2. Include a description of the vulnerability.
3. Provide steps to reproduce the issue.
4. (Optional) Suggest a potential fix.

We will acknowledge your report within 48 hours and keep you updated on the progress of the fix.
