# AmfaTools Suite & SDK 🚀
![Solana](https://img.shields.io/badge/Solana-Blockchain-blue?style=for-the-badge&logo=solana)
![NodeJS](https://img.shields.io/badge/Node.js-v18+-green?style=for-the-badge&logo=node.js)

A high-performance Solana monitoring suite. This repository provides standalone Telegram bots and a modular SDK to track liquidity, trending tokens, and security locks.

## 📦 What's Inside?
- **Raydium Monitor** (`bot-raydium.js`): Detect New V4 & CPMM Pools.
- **Pump.fun Tracker** (`bot-pumpfun.js`): Real-time trending alerts.
- **Streamflow Scanner** (`bot-streamflow.js`): 1-year lock detection.
- **Dev Scanner** (`bot-dev-scanner.js`): Wallet project history.
